Creating a booklet

Problem

I would like to create a booklet, can ptx2pdf do that for me?

Solution

Ptx2pdf does not provide this functionality. It's recommend to format the document as individual pages, then post-processing the PDF. There are a few tools you could use:
  1. Typeset the the pdf (without cropmarks!) and print it using Adobe Readers "booklet option". (See the "Booklet printing" option of the Page Scaling pop-up in the Print dialog).
  2. Typeset the the pdf (without cropmarks!) and use pdfbklt, a perl-script or pdfbklt.exe. (See http://www.sil.org/~hosken/Utils/PDFCreate.html, and ignore all the stuff about setting up Windows drivers to create PDF, as you've already got PDF output).
  3. There's a LaTeX package "pdfpages" that can do this; see the description of the "signature" option in the pdfpages documentation. (This requires a regular LaTeX installation, though, not a minimal Scripture-only XeTeX setup.)
